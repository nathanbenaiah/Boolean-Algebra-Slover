## Boolean Algebra Solver

A web-based Boolean algebra solver with interactive tools.

Features

Boolean Algebra Solver: Step-by-step expression simplification
Number System Converter**: Binary, Decimal, Octal, Hexadecimal conversion
- **Karnaugh Map Tool**: Interactive K-maps for expression minimization
- **Truth Tables**: Complete analysis with minterms and maxterms
- **Logic Circuits**: Visual gate diagrams
